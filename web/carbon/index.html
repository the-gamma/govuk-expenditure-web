Our world in data

Here, we look at a couple of simple data visualizations about our world, inspired by Our World in Data. We use The World Bank data as our data source. All of the visualizations are open and transparent. This means that you can click on the "open source button" to explore and modify the source code behind each visualizations.

For example, in the CO2 emission visualizations below, try changing the indicator from total CO2 emissions in kilotons to relative CO2 emissions per capita and see how the results change!

You can change the indicators and countries as you wish by editting the source code for the visualization if you click "open source code". However, having the source code and nicely structured data source also makes it possible to generate a simple user interface, such as the one below, where you can change the most important parameters.
